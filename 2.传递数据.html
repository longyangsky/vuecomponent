父组件原始值:{{num}}
直接传递字符串,props="传递值"
直接绑定父组件数据,实现双向绑定,:props="数据名"
实现props数据校验证String Number Boolean Array Object Function, 注意数据验证必须使用v-bind方式 
                props:{
                    name:{
                        type:String,
                        default:function(){
                            return '默认名称';
                        }
                    },
                    age:{
                        type:Number,
                        required:true
                    }
                }
